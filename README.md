## 📁 Repository Structure

- `.github/workflows/` → GitHub Actions workflows for syncing data and automation  
- `cloud_function/` → GCP cloud functions for data processing and model pipeline  
- `data/` → Structured datasets used for training and evaluation  
- `model/` → Model training and improved model implementation  
- `model_outputs/` → Model performance plots, PDPs, and feature importance outputs  
- `results/` → Prediction outputs generated over time via pipeline
